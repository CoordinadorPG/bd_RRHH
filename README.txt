# Proyecto Flask + PostgreSQL: Dashboard Operativo

## Requisitos

- Python 3.8 o superior
- PostgreSQL (con base de datos `bd_datos` ya creada)
- pip install -r requirements.txt

## Variables de entorno (o ajusta directamente en app.py)

- DB_HOST
- DB_PORT
- DB_USER
- DB_PASSWORD
- DB_NAME
